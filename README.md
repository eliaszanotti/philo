# Philosophers

## Error codes 

- 1 : Wrong amount of arguments (expected 4 or 5)
- 2 : Arguments are not valid integers ("-1" or "lorem" are invalid arguments)
- 3 : Incorrect arguments (example : more than 256 philosophers)
- 9 : Malloc cannot be created
